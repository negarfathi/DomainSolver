# DomainSolver
The DomainSolver tool generates a domain-oriented test suite to detect latent faults revealed with a specific combination of input data.

### Related Publication:
A. Kalaee, S. Parsa, and **N. Fathi**, “COSMOS: A Comprehensive Framework for Automatically Generating Domain-Oriented Test Suite,” *Information and Software Technology*, vol. 154, p. 107091, Feb. 2023, doi: 10.1016/j.infsof.2022.107091. *([Link](https://authors.elsevier.com/c/1g2Ov3O8rCcj9I))*
